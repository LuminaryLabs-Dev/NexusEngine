## Root Lesson
- id: deep-bug-root-2026-06-23-composer-read-model-handoff
- statement: `createGameKitComposer()` needs immutable nested state and `createRealtimeGame({ composer })` needs supplied-composer parity checks before composer output can be trusted as proof of resolved runtime/DSK installation.
