## Root Lesson
- id: deep-bug-root-2026-06-22-dsk-extension-install-contract
- statement: `extendDomainServiceKit()` needs end-to-end API/token parity, install atomicity, and ECS definition-name duplicate checks before DSK extension kits can be trusted as promoted service expansions.
