# Knowledge Nodes: deep_bug_report_scout 2026-06-22T18:49:24-04:00

## Root Lesson
- id: deep-bug-root-2026-06-22-host-public-state-mutation
- statement: `Nexus.Host` needs private host capability and adapter record ownership before graph snapshots can be trusted as proof surfaces.
- why it matters: Current smoke tests pass, but live probes show `host.provides` can be mutated to satisfy forged dependencies, public `adapterRecords` can inject unvalidated graph adapters, record state can be rewritten without lifecycle transitions, and throwing mount callbacks can leak host mutations without a failed record.
