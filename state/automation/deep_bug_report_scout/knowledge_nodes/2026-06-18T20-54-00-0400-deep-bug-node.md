## Root Lesson
- id: deep-bug-root-2026-06-18-boot-and-frame-order
- statement: NexusEngine's next production hardening pass should treat boot staging and frame-source ownership as shared invariants, not isolated kit bugs.
- why it matters: Valid SequenceNode graphs can become invalid when mounted before kit type registration, and frame-driven graphs can advance multiple times per engine tick when lifecycle surfaces are bound alongside direct engine driving.
